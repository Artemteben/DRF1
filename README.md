## Установка

1. **Клонируйте репозиторий:**
     git@github.com:Artemteben/DRF1.git

2. **Создайте виртуальное окружение и активируйте его:**
    ```sh
    python -m venv venv
    venv\Scripts\activate
    ```

### Запуск проекта:

Для запуска проекта выполните:
```sh
 python manage.py runserver    
```
